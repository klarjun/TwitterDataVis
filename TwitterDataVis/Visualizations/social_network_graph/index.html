Social Network graph

  • Shows how and where tweets are retweeted among different users in a directional graph.
  • Each node represent a user id, and directed edges (arrowed links) from connected the retweet_from to user_id .
  • Blue nodes represents those users who believed tweet was legit.
  • Red nodes represents those users who believed tweet was not legit.
  • Black nodes represents the retweet_from information.
  • The mouse cursor hover over each node (i.e., the user), shows corresponding user_name to the user id.
  • The single mouse click on each node enlarges them to better enhance the focus and view-ability of nodes
  • Double click bounces them back default size.
  • To find different or better orientation of nodes, keep refreshing the page untill you find a proper ditribution.
